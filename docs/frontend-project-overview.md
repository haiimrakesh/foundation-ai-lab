# Frontend Project Overview

## Purpose

The frontend is the user-facing chat experience for the Foundation AI Lab.

## Current capabilities

- Switch between six agent types
- Send chat messages to the backend
- Display assistant responses in the conversation view
- Show a payload preview of the active conversation

## Structure

- [src/App.tsx](../Unified-Chat-Application/src/App.tsx) contains the main chat experience.
- [src/main.tsx](../Unified-Chat-Application/src/main.tsx) mounts the app.
- [src/styles.css](../Unified-Chat-Application/src/styles.css) contains the current styling.

## Development notes

Run the app with the FastAPI backend at http://localhost:8000.
